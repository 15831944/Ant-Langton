Программа обрабатывает командную строку в формате
version;posX,posY;limit;success_printed;fail_printed;version_score_thread
, где
version - режим(версия) работы
posX,posY - позиция (на сетке, плоскостная система координат; X - увеличение слева направо, Y - снизу вверх) начала проверки
limit - ограничение по сумме
success_printed - признак вывода отладочной информации при проверке выполнения условий ограничения (успех)
fail_printed - признак вывода отладочной информации при проверке выполнения условий ограничения (ошибка)
version_score_thread - признак необходимости распараллеливания основного потока программы (является значимым только для режима 2(SCORE), в остальных случаях игнорируется )

ВАЖНО!
Обязательны все элементы аргумента. При отсутствии одного из них программа прекращает работу с сообщением об ошибке.

без аргумента - работа со значениями по умолчанию:
2;1000,1000;25;0;0;0
2 - обход ячеек
1000,1000 - координаты [1000,1000]
25 - ограничение суммы
0(;0) - сообщения результатов проверок точек - не выводить
0 - создать только один поток для расчета

возможные режимы работы:
0 - тест позиции на выполнение условий ограничения
1 - поиск пути (имитация движения)
2(1) - по умолчанию (обход ячеек) без распараллеливания
2(2) - обход ячеек с распараллеливанием
3 - определение множеств по граничным(реперным) точкам, нахождение их пересечения

ВАЖНО!
1 - реализация не завершена, результат=0
2(2) - завершается с ошибкой, не решены вопросы количества запускаемых потоков, результат не достоверный
3 - реализация не завершена, результат=0

примеры командной строки:
2;1000,1000;25;0;0;0 - выполнить задачу обходом ячеек
	, для позиции [1000, 1000]
	, с ограничением суммы 25
	, не выводить сообщения о результате проверок
	, без распараллеливания
2;0,0;12;0;1;0 - выполнить задачу обходом ячеек
	, для позиции [0, 0]
	, с ограничением суммы 12
	, выводить сообщения о результате проверок только об ошибках
	, без распараллеливания
2;11,2012;14;0;1;0 - выполнить задачу обходом ячеек
	, для позиции [11, 2012]
	, с ограничением суммы 14
	, выводить сообщения о результате проверок только об ошибках
	, без распараллеливания
0;12,13;9;1;0;0 - проверить выполнение условий ограничения для позиции [12, 13]
	, с ограничением суммы 9
	, выводить сообщения о результате проверок только об успехе

Описание фалов проекта:
targetver.h, stdafx.h, stdafx.cpp - служебные файлы проекта
AntCell.cpp - точка входа
Arg.h, Arg.cpp - обработка аргумента командной строки
Coord.h, Coord.cpp - описание элемента координат (оси абсцисс, ординат)
Counter.h, Counter.cpp - реализация алгоритмов подсчета
Position.h, Position.cpp - описание позиции на координатной сетке

Основные параметры для компиляции/сборки:
 - MS VS 2015 Community (v140)
 - символы: unicode
 - стандартные библиотеки Windows
 - целевая платформа Windows 8.1
 - без поддержки CLR
 - оптимизация - нет/отключено
 - библиотека времени выполнения: многопоточная отладка DLL (/MDd)
 - проверка меньших типов - Нет
 - проверка безопасности - да
 - предварительная компиляция заголовков - да
